# One key, both directions

`Ctrl+Alt+N` (`Cmd+Alt+N` on macOS) is a **focus toggle**:

- In the editor → jumps into the Nyx composer.
- Inside Nyx → jumps back to your code.

It works in both homes of Nyx — the side bar and the editor tab.

If the panel ever goes missing, the **Nyx** entry in the status bar or *Nyx: Reset Panel Location* brings it back.
