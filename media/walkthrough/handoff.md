# Hand work off — both ways

**Nyx → Cursor:** run *Nyx: Copy Chat as Markdown (Handoff)*. The whole session (messages, tool summaries, Q&A) lands on your clipboard — paste it into Cursor's agent, a PR description, or an issue.

**Cursor → Nyx:** copy any conversation or notes, then run *Nyx: Import Handoff from Clipboard*. The text rides along as context with your next message, and Nyx continues the work locally.

Typical split: let the cloud agent plan or review, let Nyx execute privately on your own hardware — or the other way around.
